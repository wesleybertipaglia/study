# Boolean

A JavaScript Boolean represents one of two values: true or false.

## Boolean()

You can use the Boolean() function to find out if an expression (or a variable) is true:

```js
Boolean(10 > 9)

10 > 9 // Or even easier:
```

## Everything With a "Value" is True

```js
100 // true
3.14 // true
;-15 // true
```

## Falses

```js
let a // false
let b = false // false
let c = null // false
let d = 0 // false
let e = -0 // false
```
