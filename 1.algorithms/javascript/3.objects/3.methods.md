# Methods

```js
let circle = {
    radius: 5,
    calculateArea: function () {
        return Math.PI * this.radius ** 2
    },
}

circle.calculateArea()
```

## Accessing Methods

```js
circle.calculateArea()
```

## Adding Methods

```js
person.name = function () {
    return this.firstName + ' ' + this.lastName
}
```
