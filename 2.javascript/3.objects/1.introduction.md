# Objects

## Creating objects

```js
// literal notation
let person = {
  name: "Wesley",
  age: 24,
  isStudent: true,
};

// constructor
let car = new Object();
car.make = "Toyota";
car.model = "Camry";
car.year = 2020;

// constructor passing properies
let newCar = new Object("Toyota", "Land Cruiser", 2024);
```

## Nested Objects

```js
let student = {
  name: "Alice",
  info: {
    grade: "A",
    age: 25,
  },
};
console.log(student.info.grade); // Output: A
```
